# 🏨 Hostel Management System (Java Swing + MySQL)

A desktop-based **Hostel Management System** built using **Java (Swing GUI)** and **MySQL**.  
This application helps manage rooms, students, and their allocation efficiently.

---

## 🚀 Features

### 🏠 Room Management
- Add new rooms
- Activate / Deactivate room
- Update room details
- Delete rooms
- View all rooms in a table
- Track room status (Booked / Not Booked)

### 👨‍🎓 Student Management
- Add new student details
- Assign available rooms dynamically
- Auto-fetch only **available rooms**
- Prevent booking if rooms are full

### 🔍 Search & Control
- Search room by number
- Prevent editing of booked rooms
- Real-time UI updates with database sync

---

## 🛠️ Tech Stack

- **Java** – GUI development  
- **MySQL** – Database  
- **JDBC** – Database connectivity  
- **NetBeans IDE** – UI design
